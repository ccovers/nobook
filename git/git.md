# GIT

## git命令
```
1、复制git项目
git clone https://github.com/ccovers/nobook.git

2、将更新提交到本地
git add .

3、添加对提交的备注
git commit -m '提交备注'

4、将本地提交推送到远端（分支）
git push
git push origin 'branch'

5、从远端（分支）拉取最新版本
git pull
git pull origin 'branch'


6、从远端拉取分支
git fetch origin 'branch':'branch'
git fetch origin

7、切换分支
git checkout 'branch'

8、备份当前分支到新的分支
git branch 'new branch'

9、回滚当前分支到指定的版本
git reset --hard 'commit id'

11、删除远程分支
git push origin :'branch'

```

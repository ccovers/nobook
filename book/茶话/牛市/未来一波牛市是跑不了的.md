# 未来一波牛市是跑不了的 [2020/6/8]
[url]: (https://t.zsxq.com/QjqNRfI)

2015年牛市历程

很多这几年新入市的朋友并不知道牛市的过程是如何发展的，今天我给大家复盘一下2015年牛市的全过程。

看图，第一个过程，2014年7月份到11月份期间。

2014年7月份，指数第一次放量突破年线，结束了一年多的底部震荡格局，成交量大约放大了三分之一。

后面就进入牛市初级阶段，但这时候很多人都还在犹豫，不敢入市，就像现在还有很多人在看空一样。

这个阶段的特征，很多股票还是处于反复震荡的状态，但是底部成交量稳步放大了，大量优质股都处于放量滞涨的形态，这种形态其实就是主力打压吸筹的阶段。

低位放量滞涨，就是主力吸货特征，前面讲过许多次了，不懂的多看精华帖。

现在指数也突破年线了，但是成交量还没有明显放出来，如果后面站稳年线，并且明显放量，平均比前面放大三分之一左右，其实只需要跟春节后站上年线的时候持平就足够了。

春节后站上年线的那段时间，要不是因为全球爆发疫情的拖累，其实就已经是牛市的开始。

牛市第一阶段，很多股票都是底部放量滞涨的形态，大家仔细去观察，现在就有很多股票已经是低位放量滞涨的形态，只不过因为今年外部大利空太多，导致低位反复震荡的时间被拉长了而已，但有些股票主力介入已经很深入，随时都可以拉升了。

等到了牛市第二阶段，那就是加速上涨阶段了。

2014年11月份开始就进入了牛市第二阶段，成交量再次大幅放大，当时就是开始鼓励险资，信托资金入市的时期，很多保险资金大举加杠杆入市，到处举牌收购上市公司，很多私募基金通过信托通道入市，有些甚至还加了三倍杠杆，也就是那种三倍劣后资金的伞形信托入市。

每天都有机构成立新基金入市的消息，成交量也大幅上升，股价也不断攀升。

等到第二阶段结束，成交量再次缩小，市场走向调整。

再到后来，很多人开始融资加杠杆入市，带动了市场走出了主升浪行情，大量加杠杆也造就了一波疯牛的行情，市场一路都不调整，就是一直涨，几乎连五日线都没有跌破过，涨到你怀疑人生。

到了第四阶段，进入了全民炒股阶段，大妈都开始进场了，大家都觉得市场再无风险，要冲上六千点，创历史新高，到八千点，一万点的时候，牛市戛然而止，后面股灾随之而来。

其实当时从技术指标也很容易看得出来，当时日线顶背离都已经确认了。

后面的股灾非常惨烈，很多股票每天开盘就是一字跌停，半个月之内就跌掉了三分之二，根本逃不出来，整个A股陷入了流动性陷阱，最后还是国家队大举入市抄底，才触底反弹的。

每天无量跌停，原因就是大量的配资盘被动平仓导致的，配资和融资爆仓的时候，配资公司都是每天开盘就挂跌停板排队卖出的，所以每天股票都是开盘就跌停，反而谁都跑不掉。

回到现实中来，现在上证指数刚刚站上年线，这是牛市开始的其中一个信号，但是还不够，还需要放量配合，现在成交量还没有明显放大，所以还不能确认牛市的开始。

只有后面成交量开始明显放大，至少放大到跟春节后那段时间持平的水平，就可以确认牛市开始了，后面就是大举做多的时候。

现在没有放量，这就是我维持谨慎做多的原因。

今晚又看到银保监会再次鼓励险资和信托资金入市的消息，让我有种似曾相识的感觉，现在的基本面环境跟2014年牛市启动之初很像，唯一不同的就是还没有出现明显放量。

如果后面大家看到各路资金开始大量入市，成交量开始明显放大，那么就要大胆做多，不要犹豫了。

话说回来，即使牛市还没开始，现在也是底部震荡行情，底部震荡行情保持半仓左右，进可攻退可守，就算一时赚不到钱，风险也并不大。

现在货币泛滥，政策鼓励，股市处于历史底部区域，未来一波牛市是跑不了的。
